# 用户故事与验收

## US-01 快速记录

作为用户，我希望选择日期后快速增加手冲次数，以便记录不打断当天节奏。

- `+1` 一次点击完成本地保存。
- 同日再次点击只更新同一条记录。
- 日历和统计立即刷新。

## US-02 明确没冲

作为用户，我希望把某天记为 0 次，以区别于忘记记录。

- 0 次在数据库中保留一行。
- 日历显示“没冲”状态，不能与空白未填写相同。
- 统计明细显示 0，但次数和手冲天数不增加。

## US-03 修正记录

作为用户，我希望修改历史日期或清除误记录。

- 支持 `-1` 和直接输入非负整数。
- 次数不会低于 0。
- 清除后该日期恢复未填写。

## US-04 回看统计

作为用户，我希望按周、月、年和全部历史查看手冲次数与天数。

- 汇总、表格和日期明细使用同一范围。
- 当前周期标注进行中。
- 未来数据不预测。

## US-05 离线可靠

作为用户，我希望没有网络也能记录和查看。

- 飞行模式完成所有核心任务。
- 重启应用后数据仍存在。
- 应用不要求登录。
