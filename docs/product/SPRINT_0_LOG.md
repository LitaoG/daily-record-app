## 本轮已冻结的决策

- 先离线 MVP，再账户与跨设备同步。
- 日历是主入口；主导航为日历、统计、活动、设置。
- 状态与数量分离；清除和明确没做不是同一动作。
- 没有目标分母前不展示完成率。
- 归档活动保留历史，P0 不强调永久删除。
- 开源项目只作为产品证据和交互参考，不直接复制 GPL 代码。
- 方向 1“暖色纸感日志”成为视觉基线。
- 手冲在导航外的活动语义中统一使用飞机图标。
- 日历不按活动无限堆点；概览采用单个汇总胶囊，精确信息进入筛选和详情。
- P0 计量类型固定为 `COUNT`、`BOOLEAN`、`DURATION`；任意日期独立颜色、目标、连续天数和提醒不进入 P0。
- Android 实现顺序固定为数据模型 `#5` → 快速记录 `#6` → 日历 `#4` → 统计 `#7`。
