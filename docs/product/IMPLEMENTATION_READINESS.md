# 实现准备度

## 已冻结

- 唯一对象：手冲。
- 每个日期最多一条记录。
- 次数为非负整数；0 表示明确没冲；无记录表示未填写。
- 主导航仅日历与统计。
- 统计周期：周、月、年、全部历史。
- 指标：总次数、手冲天数、记录日均。
- 本地 Room，P0 无登录和云同步。

## 工程门槛

- Room v2 schema 导出并纳入版本控制。
- v1→v2 迁移测试通过，旧手冲数据不丢失。
- 单元测试、Lint、Debug 编译与设备数据库测试通过。
- Figma 页面不出现活动筛选、健身或活动管理。
- 0 次、未填写、1 次、2 次和 9+ 状态完成视觉与 TalkBack 规范。

## 产品门槛

- 五人测试核心任务完成率 ≥90%。
- 常用记录中位用时 ≤5 秒。
- 统计理解率 ≥80%。
- 没有用户把 0 次误解为未填写。

- 正式展示名称尚未最终确认，暂用“手冲日历”。
- 旧通用 Figma 和 PNG 资产仍需替换后删除。
- GitHub 连接暂不可用；本地分支与提交作为唯一变更记录，恢复后再推送。
