# 界面与交互

视觉基线：暖色纸感日志、克制的陶土橙主色、飞机手冲符号。颜色只服务于手冲强度与层级，不再区分活动。

## 导航

底部仅两个入口：

1. 日历
2. 统计

版本、隐私说明等低频信息放在顶部“关于”入口，不建立复杂设置页。

## 日历页

- 顶部显示年月、上一月、下一月、回到今天。
- 日期格同时表达今天、选中状态和记录状态。
- 未填写：无标记，并提供“尚未记录”的 TalkBack 文案。
- 明确 0 次：空心短横标记与“没冲”文案。
- 1 次：浅陶土色圆点/胶囊并显示 `1`。
- 2 次及以上：随次数加深但保持可读，并显示精确数字；极端值使用 `9+` 视觉缩写，详情仍显示真实值。
- 点击未来日期只查看，不允许保存。

## 日期记录面板

- 标题为完整日期和“今天/过去日期”辅助说明。
- 中心突出当前次数。
- `-1` 与 `+1` 为主要快捷操作，数值可点击后直接输入非负整数。
- “今天没冲”把次数设为 0；“清除记录”恢复未填写。
- 保存后立即更新日历与统计，并提供短暂撤销。
- 输入错误保留用户数值并给出可修正说明，不静默截断。

## 统计页

- 顶部周期分段：周、月、年、全部历史。
- 第一屏固定三个指标：总次数、手冲天数、记录日均。
- 下方精确表格是事实展示；趋势图属于后续增强，不阻塞 P0。
- 进行中的周期明确标注；未来日期/月不预测。
- 点击表格日期返回对应日期记录面板。

## 无障碍

- 颜色不是唯一状态表达；同时使用数字、形状和文本。
- 点击目标至少 48dp。
- 支持 TalkBack、系统字体缩放与高对比检查。
- 统计图即使后续加入，也必须提供可访问表格替代。
