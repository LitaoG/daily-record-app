# 界面与交互

视觉基线：暖色纸感日志、克制的陶土橙主色、飞机手冲符号。颜色只服务于手冲强度与层级，不再区分活动。

## 导航

底部仅两个入口：

1. 日历
2. 统计

当前不建立设置或“关于”入口，避免为尚未存在的能力增加空导航。

## 日历页

- 顶部显示年月、上一月、下一月、回到今天；年月标题本身是可点击的 48dp 入口，打开年份与日期选择器。
- 可浏览范围为 1970 年 1 月至当前月；到达边界后对应月份箭头禁用。
- 快速跳转使用应用内原生暖色弹窗，不调用系统日期选择器：先点年月标题打开月历，再点年份标题进入三列年份列表，可一次跳过多年。
- 日期选择器禁止 1970 年以前和今天以后日期；切月会把选中日夹紧到目标月的合法日号，确认后只移动浏览焦点，不误打开记录编辑页。
- 快速跳转和账号同步共用 26dp 圆角、纸色背景、陶土色主按钮与 48dp 点击目标；弹窗状态用标题、文案和形状共同表达，不只依赖颜色。
- 日期格同时表达今天、选中状态和记录状态。
- 未填写：无标记，并提供“尚未记录”的 TalkBack 文案。
- 明确 0 次：显示 `0次` 与独立底色，并提供“明确记录 0 次手冲”的 TalkBack 文案。
- 1 次：浅陶土色圆点/胶囊并显示 `1`。
- 2 次及以上：随次数加深但保持可读，并显示精确数字；极端值使用 `9+` 视觉缩写，详情仍显示真实值。
- 未来日期显示“未来”并禁用点击和保存。

## 日期记录面板

- 标题为完整日期和“今天/过去日期”辅助说明。
- 中心突出当前次数。
- `-1` 与 `+1` 为主要快捷操作，次数下限为 0。
- 保存 0 次表示明确没冲；“清除记录”恢复未填写。
- 保存或清除后立即返回日历，并同步更新日历与统计。
- 数据库记录尚未读取完成时，次数控件和保存按钮保持禁用，避免把旧记录误覆盖为 0。
- 保存和清除开始前同步锁定操作，快速连点只执行一次；失败时留在当前页面、恢复按钮并显示重试提示。
- 清除记录需要二次确认；修改次数后返回需要确认是否放弃，防止误触造成数据丢失。

## 统计页

- 顶部周期分段：周、月、年、全部历史。
- 周、月、年提供上一周期、下一周期和标题日期选择器；当前周期不允许继续进入未来。
- 日历与统计共用浏览日期。切换统计周期只改变聚合范围，不重置日期：例如浏览 2026 年 5 月 6 日时，周显示 5 月 4–10 日，月显示 5 月，年显示 2026 年。
- 第一屏固定三个指标：总次数、手冲天数、记录日均。
- 下方精确表格是事实展示；趋势图属于后续增强，不阻塞 P0。
- 进行中的周期明确标注；未来日期/月不预测。
- 全部历史为空时提供“去日历记录”按钮；统计明细当前只读，不伪装成可点击入口。

## 无障碍

- 颜色不是唯一状态表达；同时使用数字、形状和文本。
- 点击目标至少 48dp。
- 支持 TalkBack、系统字体缩放与高对比检查。
- 200% 字体下月份标题改为两行，日期格增高，统计指标卡改为纵向排列，避免文字截断。
- 统计图即使后续加入，也必须提供可访问表格替代。
