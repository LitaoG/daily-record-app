# 手冲统计口径

所有统计都来自 `hand_brew_records`，不在 UI 中二次推断。

## 指标

```text
totalCount = SUM(brewCount)
brewDays = COUNT(DISTINCT localDate WHERE brewCount > 0)
averagePerBrewDay = totalCount / brewDays
```

`brewDays = 0` 时日均显示 `0.0`。`brewCount = 0` 的日期保留在明细中，但不增加次数和手冲天数。

## 周期

- 周：星期一 00:00 对应的本地日期至下周一，不跨周期重复。
- 月：月初至下月月初。
- 年：1 月 1 日至下一年 1 月 1 日。
- 全部历史：最早记录日至今天；没有记录时为空状态。

当前未结束周期显示“进行中”。未来日期和未来月份不生成预测值。

周、月、年都由同一个浏览日期 `anchorDate` 决定。切换日历月份或使用日期选择器会更新该日期；切换周/月/年标签不会把它重置为今天。历史周期显示“已结束”，当前周期才显示“进行中”或“截至今天”。

## 展示

- 周视图：固定周一到周日，以七日分布卡逐日直接标注次数和日号；过去未填写显示“未填”，明确 0 次显示 `0`，未来日显示“未来”，TalkBack 仍读出完整日期、次数和天数。
- 月视图：按自然周汇总次数和手冲天数，以横向分布条直接显示各周相对次数；行标题同时显示月内精确日期范围，过去整周无记录显示“未填写”，完全位于未来的周显示空值。切换月份必须重新构建周桶，不保留前一月行或数值。
- 年表：12 行，每月次数、手冲天数；过去整月无记录显示“未填写”，未来月份显示空值而不是预测。
- 全部历史：按年份汇总次数和手冲天数。

汇总卡、分布图和表格必须使用相同日期范围；明细加总必须等于汇总。分布图只表达已发生事实，不做预测。

## 边界测试
